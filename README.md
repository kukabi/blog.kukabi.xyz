### blog.kukabi.xyz

#### Prerequisites

Install [Hugo](https://gohugo.io/installation) extended edition. Then:

```bash
git clone --recurse-submodules https://github.com/kukabi/blog.kukabi.xyz.git
cd blog.kukabi.xyz
```

#### Local Run

```bash
hugo server
```

#### Build

```bash
chmod +x ./build.sh
./build.sh
```

The content will be under `public`.
